# FreedomIQ

FreedomIQ is a Financial Intelligence Platform designed to help investors analyze their investment portfolio and make informed decisions.

The objective of FreedomIQ is **not** to recommend what to buy or sell, but to provide comprehensive portfolio analysis, performance tracking, risk assessment, and financial independence monitoring.

---

## Current Features

- Portfolio Summary
- Performance Analysis
- Top Performers
- Top Losers
- Sector Allocation
- Portfolio Health Score
- Rebalancing Analysis

---

## Planned Features
